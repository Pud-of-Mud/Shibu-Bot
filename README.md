# Discord Rename Bot
A Discord bot that is for renaming users within a discord server. 

## Getting Started

To get set up, you'll need to follow [these bot account setup instructions](https://discordpy.readthedocs.io/en/stable/discord.html),
and then copy the token for your bot and add it into the "" varable. 

## FAQ

* If you get the following error message while trying to start the server: `429 Too Many Requests` (accompanied by a lot of HTML code), try the advice given in [this](https://stackoverflow.com/questions/66724687/in-discord-py-how-to-solve-the-error-for-toomanyrequests) Stackoverflow question.
